🐝 SnipHive for JetBrains

A secure, end-to-end encrypted code snippet manager for JetBrains IDEs.

Access your SnipHive code snippets directly in your JetBrains IDE with full E2EE support. Create, browse, and insert snippets without leaving your development environment.

Get Started at sniphive.com

✨ Features

Tool Window

Browse and search your snippets in a dedicated tool window with powerful filtering options.

Create from Selection

Instantly create snippets from selected code with Shift+Alt+S.

Insert Snippets

Insert snippets via action menu, keyboard shortcut, or intelligent code completion.

End-to-End Encryption

Client-side encryption using RSA-4096 OAEP + AES-256-GCM. Your keys never leave your device.

🔐 Security

🛡️ Enterprise-Grade Encryption

SnipHive uses industry-standard end-to-end encryption:

  • RSA-4096 OAEP for secure key exchange
  • AES-256-GCM for content encryption with authenticated encryption
  • PBKDF2 with 600,000 iterations for key derivation (OWASP recommended)
  • Your encryption keys never leave your device
  • Server never sees your plaintext content

🚀 Getting Started

  1. Install the Plugin: Go to Settings → Plugins and search for "SnipHive"
  2. Create an Account: Visit sniphive.com to sign up
  3. Login: Open the SnipHive tool window and click "Login"
  4. Setup E2EE (Optional): Go to SnipHive → Setup E2EE to enable end-to-end encryption
  5. Start Creating: Select code and press Shift+Alt+S to create your first snippet!

🔍 Usage Examples

Creating a Snippet

  • Select code in your editor
  • Right-click and choose Create Snippet
  • Enter title, select language and tags
  • Click Create

Inserting a Snippet

  • Position cursor where you want to insert
  • Right-click and choose Insert Snippet... or press Shift+Alt+I
  • Search and select a snippet
  • Click Insert

Code Completion

  • Start typing a snippet title (2+ characters)
  • Select from the completion suggestions
  • The snippet content is automatically inserted

⚙️ Configuration

Access plugin settings via Settings → Tools → SnipHive:

  • API Configuration: Custom API URL for self-hosted deployments
  • Workspace: Select your workspace (if you have multiple)
  • Display Options: Show/hide encrypted content placeholders
  • Auto-Refresh: Configure automatic refresh interval
  • Code Completion: Configure completion behavior

🔒 Anonymous Sharing Security

⚠️ Important Security Notice

When copying public URLs for encrypted snippets:

  • The URL contains an encryption key (DEK) in the hash fragment
  • Anyone with the URL can decrypt and view the content
  • Share via secure channels only (avoid email, chat)
  • URL will be visible in recipient's browser history

🔒 Privacy & Data

  • End-to-End Encryption: Your encrypted snippets are stored on SnipHive servers, but the server cannot decrypt them
  • Local Storage: Credentials and keys are stored in your IDE's Password Safe (encrypted local storage)
  • No Tracking: No usage analytics or telemetry
  • Open Source: The plugin code is open source for security audits
